# Upgrade

## This API is to trigger an upgrade/readinesscheck on a list of devices or just push an upgrade file to a list of devices.

**Type:** object

## Properties
* autoUpgradeCancel
* metadata
* description
* type
* targets
* version
* upgradePackage
* readinessCheckOnly
* enableUpgradeRevert
* name
* links
* id
* pushUpgradeFileOnly
